# Michael's Portfolio

Personal portfolio site built with plain HTML, CSS, and vanilla JS. Hosted on GitHub Pages.

## File Structure

```
index.html        → Home page (hero + previews)
projects.html     → Full project listing
experience.html   → Education, work, and skills
about.html        → Bio, values, and interests
contact.html      → Contact info + form
style.css         → Shared stylesheet
script.js         → Nav effects, mobile menu, scroll animations
.nojekyll         → Tells GitHub Pages to skip Jekyll
```

## Setup

1. Create a GitHub repository named `yourusername.github.io`
2. Push all files to the `main` branch
3. Your site will be live at `https://yourusername.github.io`

## Customization

- Replace lorem ipsum text with your real content
- Swap the gradient `.hero-avatar` / `.about-photo` with real images
- Update contact links (email, LinkedIn, GitHub URLs)
- Replace placeholder project images with screenshots
- Hook up the contact form to [Formspree](https://formspree.io) or a similar service
- Add a resume PDF to the repo and link it from the experience page
